# 核心第三方服務映像與環境變數

僅列出部署映像、部署側環境變數和後端連線環境變數。`<...>` 由部署環境提供。

## 已有基礎設施（已部署時僅供參考）

以下服務如果已經部署，只需使用實際連線地址和帳號替換後端側參數，無需重複部署。

### PostgreSQL

Keycloak、OpenFGA、Temporal、LiteLLM 也會複用這組 PostgreSQL 連線參數。

```env
# 映像
POSTGRES_IMAGE=pgvector/pgvector:0.8.5-pg18-bookworm

# 部署側
POSTGRES_INTERNAL_HOST=<postgres-host>
POSTGRES_INTERNAL_PORT=5432
POSTGRES_BIND_HOST=127.0.0.1
POSTGRES_HOST_PORT=25432
POSTGRES_USER=<postgres-user>
POSTGRES_PASSWORD=<postgres-password>
POSTGRES_DB=nexus_pro

# 後端側
DB_HOST=<postgres-host>
DB_PORT=5432
DB_USERNAME=<postgres-user>
DB_PASSWORD=<postgres-password>
DB_NAME=nexus_pro
DB_SSLMODE=<require>
```
